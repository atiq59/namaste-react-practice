# Namaste React Practice

A collection of React practice projects built with Vite. Each folder contains a mini-project demonstrating different React concepts and patterns.

## Projects Overview

### 01-counter
A simple counter application with increment, decrement, and reset functionality using React hooks.

### 02-accordion
An accordion component that expands/collapses sections with smooth transitions and icon indicators.

### 03-auto-save-text
Auto-saving text input component that persists data to localStorage and restores it on page load.

### 04-array-to-zigzag-converter
Converts comma-separated strings by reversing alternating elements in a zigzag pattern.

### 05-even-or-odd
A number checker that determines if a given number is even or odd with loading state.

### 06-chips-input
A chips/tags input component where users can add and remove tags by pressing Enter.

### 07-tab-switcher
A tabbed interface component with multiple tabs that can switch between different content sections.

### 08-progress-bar
A dynamic progress bar with +/- 10% controls and color changes based on progress level.

### 09-leap-year
A leap year checker that validates if a given year is a leap year.

### 10-copy-text
Copy to clipboard utility that copies user-entered text with success/error feedback.

### 11-authentication
A React authentication system demonstrating login/logout functionality using Context API for global state management.

## Tech Stack

- **React** - UI library
- **Vite** - Build tool
- **CSS** - Styling
- **localStorage** - Data persistence (where applicable)
- **React Icons** - Icon library (for accordion)
- **Context API** - State management (for authentication)

## Getting Started

Each project can be run independently:

```bash
cd <project-folder>
npm install
npm run dev
```
