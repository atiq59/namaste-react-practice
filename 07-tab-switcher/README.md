# Tab Switcher

A tabbed interface component that allows users to switch between different content sections.

## Features

- ✅ Multiple tabs with smooth switching
- ✅ Active tab highlighting
- ✅ Dynamic content rendering based on selected tab
- ✅ Easy to customize tabs and content
- ✅ Clean and responsive design

## How to Use

1. Click on any tab button to switch sections
2. The active tab is highlighted
3. Content updates dynamically based on the selected tab
4. Default tab is "Home"

## Default Tabs

- **Home** - Welcome message
- **Profile** - User profile information
- **Settings** - Settings configuration

## Technologies Used

- React (useState hook)
- Vite
- CSS

## Installation & Running

```bash
npm install
npm run dev
```

The application will start at `http://localhost:5173` (or the next available port).

## Component Structure

- `TabSwitcher.jsx` - Main component with tab management logic
