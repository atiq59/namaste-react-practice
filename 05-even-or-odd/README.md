# Even or Odd Checker

A utility that checks whether a given number is even or odd with loading state and error handling.

## Features

- ✅ Input validation for numbers
- ✅ Even/Odd determination
- ✅ Loading state while checking
- ✅ Error messages for invalid input
- ✅ Clean result display

## How to Use

1. Enter a number in the input field
2. Click the "Check" button
3. Wait for the result (simulated 1 second delay)
4. View if the number is even or odd

## Technologies Used

- React (useState hook)
- Vite
- CSS

## Installation & Running

```bash
npm install
npm run dev
```

The application will start at `http://localhost:5173` (or the next available port).

## Component Structure

- `EvenOrOdd.jsx` - Main component with even/odd checking logic
