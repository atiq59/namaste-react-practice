# Counter

A simple counter application demonstrating React hooks (`useState`).

## Features

- ✅ Increment counter by 1
- ✅ Decrement counter by 1
- ✅ Reset counter to 0
- ✅ Real-time display of current count

## How to Use

1. Click the **Increment** button to increase the counter
2. Click the **Decrement** button to decrease the counter
3. Click the **Reset** button to reset the counter to 0

## Technologies Used

- React (useState hook)
- Vite
- CSS

## Installation & Running

```bash
npm install
npm run dev
```

The application will start at `http://localhost:5173` (or the next available port).

## Component Structure

- `Counter.jsx` - Main counter component with state management
